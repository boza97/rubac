## Description

RuBAC (Rule Based Access Control) service

## Prerequisites

You should have installed node, docker, docker-compose on your machine before runing next commands.

## Runing with docker-compose

Position in the root folder of the project and run next command:

```bash
docker-compose up -d
```

The application should be started on: localhost:3000

To stop application just run:

```bash
docker-compose down -v
```

## Runing locally

You should follow the next steps.

### Installation

```bash
$ npm install
```

### Run database with docker

```bash
docker-compose up -d db
```

When you want to stop application stop also the db with next command

```bash
docker-compose stop db && docker-compose rm -f db
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

```

## Swagger Documentation

When application is up and runing you can visit [doc page](http://localhost:3000/api)

## Test

```bash
# unit tests

$ npm run test:e2e
```
